Django Template Language

Syntax of Template variables:

{\{ variable_name \}}

My name is {{nm}}

Current date and time is {{dt}}

Syntax of Template filters:

{\{ variable_name | filter \}}

My name is {{nm}}

My name is {{nm | upper}}

My name is {{nm | lower}}

My name is {{nm | capfirst}}


Current date and time is {{dt}}

Current date is {{dt | date:'D d M Y'}}

Current date is {{dt | date:'SHORT_DATE_FORMAT'}}

Current date is {{dt | date:'DATE_FORMAT'}}

Current time is {{dt | time:'TIME_FORMAT'}}

Syntax of Template tags:

{% comment %}

{% tag %}

{% endcomment %}

Example of Template tags:

{% if nm %}
